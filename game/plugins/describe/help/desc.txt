The %xhdesc%xn command is used to set a simple description on a character, room or exit.

   %xcdesc <name>=<description>%xn

You can also set a shorter description that is used when an at-a-glance description is needed.

   %xcshortdesc <name>=<description>%xn

You can also have multiple outfits for your character, managed using the %xhoutfits%xn command.

The %xhlook%xn command is used to look around.

   %xclook <name>%xn
   %xclook%xn  (shortcut for looking at 'here')
